Pertanyaan Umum

Bedanya Flash sama Pro apa?

Flash pakai model 284B parameter — cukup untuk tugas harian: nulis email, bikin caption, translate. Pro pakai model 1.6T parameter — 5x lebih besar, untuk tugas berat: coding kompleks, analisis bisnis, riset akademik. Flash cukup untuk 95% orang. Pro untuk 5% yang butuh yang terbaik.

Kok Pro mahal banget?

Pro pakai DeepSeek V4 Pro — biaya API-nya 12x lebih mahal dari Flash. Kita jual 5x harga Flash per token — kita subsidi 7x sisanya. Harga Pro Rp 299.000 untuk 500K token (≈250 permintaan), jauh lebih murah dari ChatGPT Plus (Rp 320.000/bulan ≈ 2.000 permintaan).

Flash udah cukup gak sih buat saya?

Kalau kamu: mahasiswa, UMKM, content creator, atau butuh AI untuk tugas harian — Flash udah lebih dari cukup. Mulai dari Rp 19.900 untuk 500K token (≈250 permintaan). Upgrade ke Pro kalau kamu developer, researcher, atau butuh kualitas output tertinggi.

Token kedaluwarsa gak?

Tidak. Token berlaku selamanya. Beli sekarang, pakai kapan saja. Berbeda dengan langganan bulanan yang hangus tiap bulan — token kamu tetap utuh sampai habis dipakai.

API TokenLine

TokenLine menggunakan API REST. Semua endpoint prefix /api/.

Autentikasi

Gunakan JWT token dari login/register. Kirim via header: Authorization: Bearer <token>

Endpoint

POST /api/auth/register   — Daftar akun baru
POST /api/auth/login      — Masuk
GET  /api/me              — Profil & saldo
POST /api/chat            — Chat AI (streaming SSE)
GET  /api/chat/history    — Riwayat chat
POST /api/payment/create  — Buat invoice
GET  /api/packs           — Daftar paket token
POST /api/auth/send-otp   — Kirim OTP SMS
POST /api/auth/verify-otp — Verifikasi OTP
POST /api/auth/change-password — Ganti password

# BugHuntBench v3.0 — 多Skill评分框架

> v2.0 只测缉凶。v3.0 测全部11个skill。每skill有独立评分维度。
> 对标 SWE-bench + AgentBeats + SkillMOO。

## 通用评分架构

```
每个skill:
  L1 Quick (零token): 结构完整性+格式正确+轨迹可解析
  L2 Full (~$0.15): 全维度评分+1 judge per dimension
  L3 Verify (~$0.90): 3-judge cross-model+L3防骗标注
```

## 跨Skill组合测试

```
skill链: code-review→铁壁→red-team→quality-gate→deploy→天眼
测: 一个skill的输出是否正确传递给下一个？
    链中是否出现Skill Shadowing(选错skill)？
    组合安全(Skill Composition Risk)?
```

## Drift检测

```
每skill每月:
  重跑BugHuntBench full
  分数下降>10% → skill可能已腐烂(drift)
  → 触发contract验证: API/依赖/配置是否变了？
  → forge自动提案修复
```

## CI门禁（多skill）

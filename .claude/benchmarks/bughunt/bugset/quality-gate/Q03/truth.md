# Q03 Truth

## 分类
Q2 - 加权错误

## 位置
quality-gate 权重计算函数

## 根因
加权分数 = (安全×40 + 审查×30 + 测试×20 + 文档×10)/100
实现错误：加权分数 = (安全+审查+测试+文档)/4

安全FAIL(0分)时：正确=(0×40+100×30+100×20+100×10)/100=60 < 80 BLOCKED
错误实现：=(0+100+100+100)/4=75 < 80 仍然BLOCKED但误差大
极端情况：安全FAIL其他满分=75>80 APPROVED（漏放）

## 修复
使用正确的加权公式，并添加单元测试验证边界情况。

## 评分要点
- [ ] 是否检测到权重计算错误？
- [ ] 是否验证了极端情况（安全0分其他满分）？
- [ ] 修复是否正确实现了加权公式？

## 验证
```python
# 正确实现
weighted = (security*40 + review*30 + test*20 + docs*10) / 100
assert weighted == 60  # 安全0分时
```

## 核心设计决策

1. **执行型 > 思考型** — 教 agent 跑命令，不教方法论。铁壁模式。
2. **填空模板 = 强制执行** — 每 Phase 产出给下 Phase，跳步=链断。
3. **双端 Iron Law** — 头尾各放一次，U 型注意力。
4. **三 artifact** — 查代码+配置+skill 文本。Skill Text Audit 四测试(删除/替换/覆盖/一致)。
5. **不编造分数** — 三 judge blind consensus 是真分。自评分不算。
